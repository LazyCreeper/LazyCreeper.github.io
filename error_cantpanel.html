无法访问面板？的解决方案

  • Windows
  • Linux

Windows 用户

1.检查你的浏览器URL是否输入错误。
2.如果你是VPS，请关闭Windows的防火墙功能，并开放面板相对应的端口。
3.如果你的服务商只提供了一个端口来运行Minecarft服务器，那么你需要向你的服务商多恰一个端口来供面板访问。（记得修改面板目录下的property.js文件）
4.如果你是用的Sakura Frp，那么请将你的线路改为国外线路才能访问面板。

Linux 用户

1.跟上面一样
2.看看面板是否在运行，建议使用systemctl来管理面板。具体如何操作请点击这里。
