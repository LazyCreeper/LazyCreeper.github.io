无法访问面板？的解决方案

  • Windows
  • Linux

Windows 用户

1.检查你的浏览器URL是否输入错误
2.如果你是VPS，请关闭Windows的防火墙功能，并向你的服务商多恰一个端口来访问面板

Linux 用户

1.跟上面一样
2.看看面板是否在运行，建议使用Screen来运行面板防止关闭终端时把面板也跟着关了。具体如何操作请点击这里。
