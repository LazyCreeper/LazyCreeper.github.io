运行在Windows

方法一：

第一步：从百度网盘下载已经整合好的Mcserver Manager

第二步：解压到一个你喜欢的地方

第三步：双击Start.bat运行

第四步：打开浏览器输入127.0.0.1:23333然后输入账号：#master密码：123456就可以进入面板啦！

如果你对文字不感兴趣，你可以试试视频教程

方法二：

第一步：安装Nodejs（版本要大于8.0）

第二步：下载MCSM源代码并解压到一个你喜欢的地方: https://github.com/Suwings/MCSManager/archive/master.zip

第三步：进入程序根目录，按 Shift + 鼠标右键选择打开命令行窗口(请忽略那个npm install)

第四步：输入cd MCSManager进入MCSM目录然后执行命令npm install --production进行安装

第五步：执行命令node app.js或者npm start启动面板

最后一步：打开浏览器输入127.0.0.1:23333就可以进入面板啦！账号是#master，密码是123456


方法三：

第一步：下载并安装Nodejs和Git工具

第二步：在一个你喜欢的地方新建一个文件夹

第三步：按 Shift + 鼠标右键选择打开命令行窗口

第四步：输入命令git clone https://github.com/Suwings/MCSManager.git克隆代码

第五步：执行命令npm install --production进行安装

第六步：执行命令node app.js或者npm start启动面板

最后一步：打开浏览器输入127.0.0.1:23333就可以进入面板啦！账号是#master，密码是123456

注意，方法2和方法3的区别就是有没有使用Git工具。。。
