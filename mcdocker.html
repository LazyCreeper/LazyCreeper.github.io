使用Docker来保护你的服务器 (仅Linux)

  • 安装Docker
    • Ubuntu
    • CentOS
    • Arch Linux
  • 创建Docker
  • 创建服务端
  • 启动服务器

第一步：安装Docker

Ubuntu：

sudo apt-get update
sudo apt-get install docker.io

启动 Docker 服务（重要）

sudo service docker start

CentOS：

yum update -y
yum -y install docker

启动 Docker 服务（重要）

sudo service docker start

Arch Linux

安装 Docker

pacman -S docker

启动 Docker

systemctl start docker

开机启动 Docker

systemctl enable docker

关掉开机启动 Docker

systemclt disable docker

最后

输入以下命令，输出版本号即为正确

docker version

第二步：创建Docker

  1. 使用#master账号登录

  2. 进入 服务端管理

  3. 选择中上方的 创建虚拟镜像 中创建 Docker 镜像。

  4. 默认情况下，无需修改。点击 创建镜像

  5. 最后。等待，速度与你网络有关，任务结果列表 显示成功则成功。

第三步：创建服务端

创建服务端十分简单，点击“引导创建服务端”即可。通过里面的指引，创建你的 MC 服务器，稍后可以通过文件在线管理上传你的核心文件。

此端的 Docker 配置

  1. 创建好服务端之后，点击这个服务器实例的【参数】然后到【docker 配置】

  2. 勾上【此服务端启用 Docker 容器】选择框

  3. 在【端口限制】中填写25565:25565即可，此意是开放25565端口。冒号两边一般情况下保持一致即可。

  4. 保存

第四步：启动服务器

  1. 点击左侧菜单进入【文件在线管理】，找到该服务端并进入，然后上传你的核心文件。

  2. 启动服务器

  • 启动时，Docker 容器会自动加载你的 Minecraft 服务端，保证安全性，稳定性。
  • 如果命令控制台出现一堆乱码，使用 模拟型终端 即可
