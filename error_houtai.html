如何让面板在后台运行？

Windows用户：

保持面板的cmd不要被关就行~


Linux用户：

先安装screen软件

Ubuntu：sudo apt install screen
CentOS：yum install screen

然后输入screen -S mcsm创建一个名字叫做mcsm的窗口
最后进入到你的面板目录，输入npm start启动你的面板
如果与终端断开连接，可以使用screen -r mcsm重新连接
