一行命令快速安装

适用于amd64架构 Ubuntu/Centos/Debian/Archlinux

wget -qO- https://gitee.com/Suwingser/MCSManager-installer/raw/master/install.sh | bash

运行在Ubuntu（这里使用Win子系统Ubuntu演示)

第零步：输入sudo -i切换到root用户

第一步：输入apt update更新包

第二步：分别执行以下命令安装 Nodejs

curl -fsSL https://deb.nodesource.com/setup_16.x | bash
apt install -y nodejs

第三步：输入apt install -y git安装git

第四步 [ 可选 ]：输入apt -y install npm安装npm

第五步：输入：git clone https://github.com/Suwings/MCSManager.git克隆源码

第六步：输入cd MCSManager进入程序根目录，然后执行命令npm i，如果提示-bash: npm: command not found就去执行第四步

第七步：执行命令node app.js或者npm start启动面板

最后一步：打开浏览器访问 http://你的服务器地址:23333 然后输入默认账号：#master密码：123456就可以进入面板啦！

如果你还是不会，可以考虑我的付费安装服务

运行在Centos

第一步：用root用户连接到你的CentOS服务器

第二步：输入yum -y install curl安装curl工具。

第三步：输入curl -sL https://rpm.nodesource.com/setup_16.x | bash

第四步：输入yum install -y nodejs安装NodeJs 16。

第五步：输入node -v检查Nodejs是否安装成功（出现版本号即为安装成功）

第六步：输入yum -y install git安装git

第七步：输入git clone https://github.com/Suwings/MCSManager.git克隆源码

第八步：输入命令cd MCSManager进入程序根目录，然后执行命令npm i安装依赖

第九步：输入npm start或者node app.js就能启动面板了

第十步：打开浏览器访问 http://你的服务器地址:23333 就能进入面板了。如果是本机测试就访问127.0.0.1:23333然后输入默认账号：#master密码：123456就可以进入面板啦！

在后台保持运行

先安装screen软件

Ubuntu：apt install -y screen
CentOS：yum install -y screen

然后输入screen -S mcsm创建一个名字叫做mcsm的窗口。

最后进入面板目录，然后用npm start启动你的面板。

如果与终端断开连接，可以使用screen -r mcsm重新连接。

开机自启动

注意：该方法同样也可以让面板保持在后台运行，无需使用 screen 。如果你使用的是一键安装脚本，则无需配置

第一步：创建文件/lib/systemd/system/mcsm.service

[Unit]
Description=MCSManager 											
				
[Service]
ExecStart=node /root/MCSManager/app.js
Restart=always
Environment=PATH=/usr/bin:/usr/local/bin:/usr/local/node/bin
Environment=NODE_ENV=production
WorkingDirectory=/opt/MCSManager/
				
[Install]
WantedBy=multi-user.target

第二步：输入systemctl enable mcsm设置面板开机自启动

第三步：即可使用systemctl管理面板启动与关闭

开启：systemctl start mcsm
关闭: systemctl stop mcsm
状态: systemctl status mcsm
