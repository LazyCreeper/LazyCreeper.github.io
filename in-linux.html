运行在Linux

（因为没有Linux机器所以我用Win子系统Ubuntu演示)

第零步：输入sudo -i使用root用户

第一步：输入apt update更新包

第二步：输入apt install nodejs安装nodejs

第三步：输入apt install git安装git

第四步：输入apt install npm安装npm

第五步：执行命令：git clone https://github.com/Suwings/MCSManager.git克隆源码

第六步：输入命令cd MCSManager进入程序根目录，然后执行命令npm install --production

第七步：执行命令node app.js或者npm start启动面板

最后一步：打开浏览器输入127.0.0.1:23333然后输入默认账号：#master密码：123456就可以进入面板啦！
