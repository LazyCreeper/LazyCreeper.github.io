运行在Ubuntu

（这里使用Win子系统Ubuntu演示)

第零步：输入sudo -i切换到root用户

第一步：输入apt update更新包

第二步：输入apt install nodejs安装nodejs，装完后记得输入node -v检查版本，如果没显示那就重装一次。如果版本过低请看这里

第三步：输入apt install git安装git

第四步 [ 可选 ]：输入apt install npm安装npm

第五步：输入：git clone https://github.com/Suwings/MCSManager.git克隆源码

第六步：输入cd MCSManager进入程序根目录，然后执行命令npm install，如果提示-bash: npm: command not found就去执行第四步

第七步：执行命令node app.js或者npm start启动面板

最后一步：打开浏览器输入127.0.0.1:23333然后输入默认账号：#master密码：123456就可以进入面板啦！

如果你还是不会，可以考虑我的付费安装服务


运行在Centos / RedHat

（这是一台全新的Centos机器）

第一步：用root用户连接到你的CentOS服务器 [如果是本机就用sudo -i切换到root用户]。

第二步：输入yum -y install git安装git工具。

第三步：输入yum -y groupinstall "Development Tools"安装Development Tools。

第四步 [可选] ：输入yum -y install wget安装wget工具。

第五步：输入wget https://npm.taobao.org/mirrors/node/v11.0.0/node-v11.0.0.tar.gz从淘宝镜像下载Nodejs。[如果出现-bash: wget: command not found的情况请重复第四步]。

第六步：输入命令tar -xvf node-v11.0.0.tar.gz解压安装包。

第七步：输入命令cd node-v11.0.0进入目录。

第八步：输入命令yum install gcc gcc-c++安装Gcc-c++插件。

第九步：输入命令./configure使用默认配置。

第十步：输入make进行编译。[可能需要差不多半小时]

第十一步：输入命令make install开始安装Nodejs

第十二步：输入node -v检查Nodejs是否安装成功，如果显示-bash: node: command not found请重复第十步，出现版本号即为安装成功

第十三步：git clone https://github.com/Suwings/MCSManager.git克隆源码

第十四步：输入命令cd MCSManager进入程序根目录，然后执行命令npm install --production

第十五步：输入npm start或者node app.js就能启动面板了

第十六步：打开浏览器访问 http://你的服务器地址:23333 就能进入面板了。如果是本机测试就访问127.0.0.1:23333然后输入默认账号：#master密码：123456就可以进入面板啦！

在后台保持运行

先安装screen软件

Ubuntu：sudo apt install screen
CentOS：yum install screen

然后创建一个名字叫做mcsm的窗口。

最后npm start启动你的面板。

如果与终端断开连接，可以使用screen -r mcsm重新连接。
