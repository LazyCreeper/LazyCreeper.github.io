无法访问面板？的解决方案

  • Windows
  • Linux

Windows 用户

1.检查你的浏览器URL是否输入错误
2.检查你的面板端口是否被占用 [ 默认为23333端口 ]。如果是，请关闭占用端口的程序或者打开面板目录下的property.js文件修改面板端口。
3.如果你是VPS，请关闭Windows的防火墙功能，并向你的服务商多恰一个端口来访问面板
