运行在Android

  1. 安装 Termux
  2. 更新软件包
    3. apt update
  3. 安装 Nodejs
    4. apt install nodejs
  4. 安装 npm
    5. pkg install npm
  5. 安装 git
    6. pkg install git
  6. 准备安装目录mcsm
    7. cd ~mkdir mcsmcd ~/mcsm
  7. 下载面板端（Web）程序
    8. git clone https://github.com/MCSManager/MCSManager-Web-Production.git
  8. 重命名文件夹并进入
    9. mv MCSManager-Web-Production webcd web
  9. 安装依赖库
    10. npm install --production --registry=https://registry.npmmirror.com/
  10. 返回安装目录
    11. cd ~/mcsm
  11. 下载守护进程（Daemon）程序
    12. git clone https://github.com/MCSManager/MCSManager-Daemon-Production.git
  12. 重命名文件夹并进入
    13. mv MCSManager-Daemon-Production daemoncd daemon
  13. 安装依赖库
    14. npm install --production --registry=https://registry.npmmirror.com/
  14. 然后启动守护进程（Daemon）
    15. node app.js
  15. 在屏幕左边往右滑，选择 NEW SESSION 新建一个终端
  16. 进入Web端目录
    17. cd ~/mcsm/web
  17. 启动面板（Web）
    18. node app.js

至此，面板已经安装完成了，启动浏览器访问http://安卓设备的ip地址:23333/即可进入面板
默认情况下，面板端会自动扫描 daemon 文件夹并且自动连接到守护进程。

手机 ip 地址获取方法

  1. 在设置->WLAN里面看
  2. 如果设置里面没有，可以使用第三方软件查看（比如Devcheck）
  3. 在路由器后台看

经测试，MCSManager在小米平板1、红米Note5A，酷珀K11这三台设备上运行良好
