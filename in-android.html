运行在Android

第一步：安装 Termux

第二步：打开Termux，输入apt update并按回车进行升级

第三步：输入apt install nodejs安装Nodejs

第四步：输入wget https://github.com/Suwings/MCSManager/archive/master.zip下载MCSM

第五步：输入unzip master.zip解压MCSM

第六步：输入cd MCSManager-master进入程序根目录

第七步：输入npm install --production安装(如果出现错误请先执行命令安装npmapt install npm)

最后输入npm start或者node app.js启动你的面板
