运行在Android

第一步：安装 Termux

第二步：打开Termux，输入apt update并按回车更新包

第三步：输入apt install nodejs安装Nodejs

第四步：输入apt install git安装git

第五步[可选]：输入命令pkg install npm安装npm

第六步：输入git clone https://github.com/Suwings/MCSManager.git克隆源代码

第七步：输入cd MCSManager进入程序根目录

第八步：输入npm install --production安装(如果出现错误请从第五步开始)

最后输入npm start或者node app.js启动你的面板
